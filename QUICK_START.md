# 🚀 CPG2PVG-AI 快速部署指南

## 🎯 最简单的部署方法

我已经为你准备好了**完全自动化**的部署脚本！只需要运行一个命令即可完成整个部署流程。

## 📋 一键部署

### Windows用户 (推荐)
```bash
# 在项目根目录运行
auto-deploy.bat
```

### Linux/macOS用户
```bash
# 在项目根目录运行
./auto-deploy.sh
```

## 🔧 脚本功能

自动化脚本将帮你完成：

✅ **系统检查** - 验证所有依赖已安装
✅ **项目验证** - 确认代码和配置正确
✅ **前端部署** - 自动打开Vercel部署页面
✅ **后端部署** - 自动打开Railway部署页面
✅ **数据库初始化** - 提供详细操作指南
✅ **Redis配置** - 可选缓存服务配置
✅ **部署验证** - 完整的功能检查清单
✅ **生成报告** - 自动生成部署完成报告

## ⏱️ 预计时间

- **自动化脚本运行**: 2-3分钟
- **实际部署时间**: 45-60分钟
- **包含等待和配置时间**
